MVRS Mining Guide

Mine MVRS Coin Using Termux (Android)

1. Install Termux

Download Termux from the official source:

F-Droid (Recommended)

https://f-droid.org/en/packages/com.termux/

Do NOT download from Play Store (outdated).

2. Update Termux Packages

After opening Termux, enter:

pkg update -y && pkg upgrade -y

3. Install Required Packages

Enter this:

pkg install git -y

4. Download MVRS Auto Miner

Run this command:

git clone https://github.com/MVRS-official/MVRS_auto.git

Then go into the folder:

cd MVRS_auto

5. Give Permission

Allow the miner to run:

chmod +x mvrs_node chmod +x setup_mvrs.sh

6. Start MVRS Node

Run:

./mvrs_node

You will see:

=== MultiVerse (MVRS) Mobile Node === Commands: help | mine | start_auto <mins> | stop_auto | balance | history | send <amount> <wallet_address> | exit >

7. How to Mine Manually

Inside the MVRS console:

> mine

You will receive:

Block mined: 000xxx... New balance: 50 MVRS

8. Auto Mining

Start auto miner:

> start_auto 1

(This mines every 1 minute)

Stop auto miner:

> stop_auto

9. Check Balance

> balance

10. View Mining History

> history

11. Send MVRS to Wallet

Use:

> send AMOUNT WALLET_ADDRESS

Example:

> send 100 ABCD1234WALLET9876

12. Exit Miner

> exit
